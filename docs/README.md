## 🏗 Архитектура проекта
main.py → Интерфейс командной строки
↓
services.py → Бизнес-логика (правила, проверки, запросы)
↓
tables.py → Модели базы данных (SQLAlchemy ORM)
↓
db_session.py → Подключение к SQLite

**Принцип разделения ответственности:**
- `main.py` — только ввод/вывод, работа с пользователем
- `services.py` — все операции с данными и бизнес-правила
- `tables.py` — структура таблиц и связи между ними
- `db_session.py` — техническая прослойка для подключения к БД

## 🚀 Быстрый старт

Чтобы начать пользоваться платформой ReHub, выполните следующие шаги:

1. **Регистрация**: При первом запуске введите своё имя, email и краткое описание навыков.
2. **Создание проекта**: Нажмите «Создать проект», укажите название и выберите категорию (IT, Media или Fashion).
3. **Поиск команды**: Просматривайте ленту проектов и отправляйте заявки на участие в тех, что вам интересны.
